# Тестовое задание

Для того чтобы запустить программу вам необходимо:

1. Выбрать место на жеском где вы хотели бы установить данное тестовое задание
2. Зайти в консоль разрабочика и ввести команду git clone https://github.com/ryslanukhlin/TakeoffStaff
3. Далее нужно выполнить скачивание npm пакетов, если вы не закрыли терминал можно ввести
   **cd TakeoffStaff** и далее **npm i**
4. Нужно ввести команду **npm run start** и в другом терминали в той же директории TakeoffStaff
   ввести **json-server db.json**
